# NYC Airbnb Market Analysis

## Project Overview
This project explores the Airbnb market in New York City. The goal was to clean, analyze, and visualize a dataset of over 100,000 Airbnb listings to uncover key trends in pricing, location, and host behavior across the 5 boroughs.

## Key Objectives
- **Price Analysis:** Identify the most expensive and most affordable neighborhoods.
- **Market Distribution:** Analyze the distribution of room types and availability across different boroughs.
- **Quality & Cost:** Understand the relationship between price, service fees, and review ratings.

## Key Insights
1. **Market Dominance**: Manhattan and Brooklyn completely dominate the market, holding 83% of all listings.
2. **Room Preferences**: "Entire home/apt" is the most prevalent room type (52%+) and commands the highest nightly prices.
3. **Price vs. Quality**: Surprisingly, there is a near-zero correlation between the price of a listing and its review ratings.
4. **Service Fees**: The service fee is strictly applied as a fixed 20% markup on the nightly price.

## Tools & Technologies
- **Tableau**: Interactive Data Visualization & Dashboarding
- **Python (Pandas, Matplotlib, Seaborn)**: Data Cleaning and Exploratory Data Analysis
- **Jupyter Notebook**: Development Environment

## Repository Structure
- `NYC_Airbnb_Market_Analysis.twb` - The Tableau Workbook containing the dashboard.
- `cleaned_airbnb_final.csv` - The cleaned dataset used for visualization.
- `data_cleaning.ipynb` - Python notebook detailing the 14-step data cleaning pipeline.
- `exploratory_analysis.ipynb` - Python notebook containing statistical insights and initial EDA.

## How to Run
1. Clone the repository to your local machine.
2. Open `NYC_Airbnb_Market_Analysis.twb` using **Tableau Desktop** to explore the dashboard interactively.
3. Alternatively, view the live published version on Tableau Public.
